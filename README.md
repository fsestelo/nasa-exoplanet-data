# Are We Alone in Space?

## Analyzing NASA's Exoplanet Data with Python

This project was developed as part of the **Udacity Data Analyst Nanodegree**. It performs an **Exploratory Data Analysis (EDA)** using data from NASA's Exoplanet Archive to investigate characteristics of discovered exoplanets.

The notebook demonstrates how to load, clean, analyze, and visualize scientific data using Python, helping identify trends and relationships between planetary attributes.

---

# Getting Started

These instructions will help you run the notebook on your local machine.

## Prerequisites

Make sure you have the following installed:

* Python 3.14+
* Jupyter Notebook (or JupyterLab)
* pip

---

# Dependencies

This project uses the following Python libraries:

* **pandas** – Data manipulation and analysis
* **matplotlib** – Static data visualization
* **seaborn** – Statistical data visualization
* **plotly** – Interactive visualizations
* **hvPlot** – High-level plotting API for pandas

Install all dependencies with:

```bash
pip install pandas matplotlib seaborn plotly hvplot notebook
```

---

# Installation

1. Clone this repository:

```bash
git clone https://github.com/fsestelo/nasa-exoplanet-data.git
```

2. Install the required packages:

```bash
pip install pandas matplotlib seaborn plotly hvplot notebook
```

3. Start Jupyter Notebook:

```bash
jupyter notebook
```

5. Open the notebook and run all cells.

# Project Content

The notebook completes the following tasks:

* Load NASA's Exoplanet dataset.
* Inspect dataset structure and data types.
* Generate descriptive statistics.
* Practice Python fundamentals using lists, sets, and sorting.
* Analyze the relationship between planet radius and mass.
* Explore how planet discoveries evolved over time.
* Identify the most common exoplanet discovery methods.
* Create visualizations to support data interpretation.
* Draw conclusions based on exploratory data analysis.
